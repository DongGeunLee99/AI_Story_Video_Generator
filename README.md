# 🎬 AI Story Video Generator

이 프로젝트는 **짧은 시간 안에 많은 스토리 영상을 빠르게 생성하는 것에 초점을 둔 AI 기반 영상 생성 프로젝트**입니다.

영상의 시네마틱한 퀄리티보다는, **콘텐츠를 대량으로 효율적으로 생산하는 자동화 파이프라인 구축**에 중점을 둡니다.

텍스트 기반 스토리를 입력하면, AI가 자동으로 **이미지 · 음성 · 자막** 을 생성하여

**오디오북 형태의 스토리 영상을 빠르게 생성**할 수 있도록 설계했습니다.

LLM, 이미지 생성 모델, TTS를 결합해

**스토리 → 장면 → 영상**으로 이어지는 자동화 파이프라인을 구축했습니다.

---

## 📌 프로젝트 개요

이 프로젝트는 사용자가 입력한 스토리를 기반으로 다음 과정을 자동화합니다.

- 스토리를 장면(Scene) 단위로 분할
- 각 장면에 맞는 이미지 프롬프트 생성
- AI 이미지 생성
- 스토리 또는 내레이션용 음성 생성
- BGM 삽입 및 음성 파라미터 조절
- 생성된 이미지와 음성을 결합해 영상 생성

AI 기반 콘텐츠 자동 생성 파이프라인을 실험하고,

**스토리텔링 + 생성형 AI의 대량 생산 가능성**을 검증하는 데 목적이 있습니다.

---

## 🧩 주요 기능

### 📝 스토리 입력

- 사용자가 자유 형식의 스토리 텍스트 입력
- LLM을 활용해 장면(Scene) 단위로 스토리 분해

### 🎨 이미지 생성

- 각 장면별 설명을 기반으로 이미지 프롬프트 생성
- Stable Diffusion 계열 이미지 생성 모델 활용
- 빠른 생성 속도를 우선한 이미지 생성 전략 적용

### 🔊 음성 생성 (TTS)

- 스토리 또는 장면 설명을 기반으로 음성 생성
- **음성 속도(speed) 조절 가능**
- **음성 볼륨(volume) 조절 가능**
- 영상 내레이션에 맞춘 음성 파라미터 커스터마이징

### 🎵 BGM 삽입

- 영상 분위기에 맞는 배경 음악(BGM) 자동 삽입
- 음성과 BGM의 볼륨 밸런싱 처리
- 오디오북 형태의 영상 구성 지원

### 🎥 영상 생성

- 생성된 이미지, 음성, BGM을 순서대로 결합
- 장면 전환을 포함한 단일 영상 파일 생성
- 대량 영상 생성을 고려한 자동화 파이프라인 구성

### 🧠 AI 파이프라인 통합

- LLM → Image Generation → TTS → Video 생성 흐름을 하나의 프로세스로 연결
- **Modal 기반 분산 실행 환경 사용**
